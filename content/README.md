# LinkedIn Content System

This folder manages LinkedIn content planning, drafting, and publishing for Integrated.ee, posted from Villem's personal profile.

## Folder Structure

```
content/
  strategy.md        — Content pillars, formats, cadence, voice guidelines
  calendar.md        — Rolling weekly content calendar
  pillars/           — Post ideas organized by content pillar
  drafts/            — Work-in-progress posts (use TEMPLATE.md)
  published/         — Archive of published posts (move from drafts/ after posting)
```

## Workflow

1. **Find an idea** — Browse `pillars/` files for post ideas or add new ones
2. **Draft** — Copy `drafts/TEMPLATE.md` to `drafts/YYYY-MM-DD-slug.md` and write
3. **Review** — Check against `strategy.md` voice guidelines
4. **Publish** — Post on LinkedIn
5. **Archive** — Move the draft to `published/` with the publish date

## Cadence

3x/week: Monday, Wednesday, Friday. See `calendar.md` for the schedule.

## Repurposing Blog Posts

Many post ideas come from existing blog posts in `homepage/posts/`. When repurposing:
- Extract one key insight per LinkedIn post (don't summarize the whole article)
- Add a personal angle or opinion
- Link to the full blog post at the end (optional, not required)
- Consider turning longer posts into carousels (PDF, 5-8 slides)
